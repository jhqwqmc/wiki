# :cuble_postle: 合成的

> 在同一空间渲染多个子模型。

```yaml
default:composite_item:
  model:
    type: "minecraft:composite"
    models:
      - type: minecraft:model
        path: "minecraft:item/custom/model_1"
      - type: minecraft:model
        path: "minecraft:item/custom/model_2"
      - type: minecraft:model
        path: "minecraft:item/custom/model_3"
```
