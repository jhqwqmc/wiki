# 🎨 Tint

{% hint style="success" %}
所有使用颜色的地方支持使用十进制或 RGB 格式

```yaml
价值：16711680
```

```yaml
值：
  - 255
  - 0
  - 0
# 或
值：
  - 1.0
  - 0.0
  - 0.0 - 0.0
```

```yaml
值：255,0,0
# 或
值：1.0,0.0.0.0
```

{% endhint %}
