# 🪨 坚固的基础块

```yaml
块:
  默认:peb:
    行为:
      类型: sturdy_base_block
      方向: 向下
      支持类型:
        - 已满
```

[sturdy-base-block](sturdy-base-block “提及”指需要坚固支持面部的方块。 例如，门不会在滴灌上工作。 支持类型包括`完整'、`僵硬`和`中心\`。
